State Management Web App

A server-side web application designed to demonstrate full CRUD (Create, Read, Update, Delete) functionality using JSP, JavaBeans, and a MySQL database. This project simulates a professional workflow for managing U.S. state records through a user-friendly, dynamic interface. All database operations are handled via a reusable JavaBean (StateDAO.java), promoting clean architecture and separation of concerns. The application reflects real-world web development practices with a focus on maintainability, scalability, and clarity.

Technologies: Java, JSP, JavaBeans, MySQL, HTML, CSS
